VIA3 Unity

VIA3 Unity is platform focused on connecting teams & networks, utilizing real time video conferencing, chat messaging, & presence awareness. I primarily focused on the Front End of the platform creating channels similar to that of slack with file uploading/sharing & video calls enabled. It is an Angular 10+ SPA using SignalR for realtime user interactions, C# & .NET backend, SQL database, and Auth0 user sign-in/authorization.

PHOTO FUSER

Community-driven web application where a user can upload, share, and comment on photos created using the Deep Art Effects API. This machine learning technology merges the user's photos with famous well-known images. I Created the Back End logic that displayed dynamic content made by the user and modeled the UI and UX after Instagram using SASS.
